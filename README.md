## 🚀 快速开始

### 方式一：直接访问
```bash
# 克隆仓库
git clone https://github.com/1837620622/ckstudio.git

# 进入目录
cd ckstudio

# 启动本地服务器
python3 -m http.server 8080

# 访问 http://localhost:8080
```

### 方式二：使用 Live Server
1. 使用 VS Code 打开项目
2. 安装 Live Server 扩展
3. 右键 `index.html` → Open with Live Server
